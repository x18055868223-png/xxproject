## 1. 一句话定位
估计到期前价格**首次触及 loss_boundary** 的概率。是五决策变量 #1（touch_now）与 drift 的核心，也是入场锚 `entry_touch_probability` 的算法。**风险控制估计，非真实胜率宣称**。

## 2. 当前具体实现（`estimate_touch_probability`）
```
已击穿 boundary                                  → 1.0
dte ≤ 0                                          → 0.0
IV 可用（_normalise_iv）：无漂移对数正态障碍近似
    t_years = dte/(24·365)
    sigma_t = vol·√t
    distance = ln(boundary/price)  (SHORT_CALL)  /  ln(price/boundary)  (SHORT_PUT)
    z = distance / sigma_t
    p = clamp(2·(1 − N(z)), 0, 0.98)             ← 反射原理 first-passage 近似
IV 不可用：delta 兜底
    p = clamp(|short_delta|·1.8, 0, 0.95)        ← 保守，调用方标 LOW 置信
```
- `_probability_confidence`：IV 可用 → `HIGH`，否则 `LOW`。
- `_norm_cdf` = erf 实现；`_normalise_iv`：>3 视百分比 ÷100。

## 3. 关键参数（硬编码）
反射原理系数 2.0、上限 0.98（IV 路径）；delta 兜底系数 1.8、上限 0.95。年化 365、零漂移近似（短 DTE 合理）。

## 4. 整合中的路径修改
**公式不变**（总纲 v0.3 §3.3.5）——它本就 IV-based，删 KPF 不触碰它。反而消除了"用 KPF buffer 改触界概率"的虚假安心风险（审计结论第 4 条）。
- 与 VRP 的 BS pricer 共享数学原语：`_norm_cdf`/`_normalise_iv` 是 VRP 收口契约的 canonical 目标（VRP 删本地副本复用 `hedge_risk._norm_cdf`/`_normalise_iv`，见 03_VRP门/04_BS_pricer）。

## 5. 当前目标 / 待办
- 无独立校准项（风险控制估计）；随对冲整体 12 万样例回放验证 drift/touch 阈值。
- IV 缺失走 delta 兜底并标 LOW——实盘应尽量保证 IV 可得（Deribit greeks）。

## 6. 边界与陷阱
- 这是**首次触界概率（barrier first-passage）**，不是到期 OTM 概率，更不是真实胜率——别与执行层 `plan_win_rate=1−|delta|` 混用。
- 无漂移假设：短 DTE 下合理，长期限会失真。
- delta 兜底是**保守高估**（×1.8），仅在 IV 不可得时用，且必标 LOW 置信。
