## 一、最终落地结论

以 v2.0 主体框架（统一主链 / 推荐库+硬授权 / 软授权退出 / 80% 资格-预算解耦 / 固定 BTC-PERPETUAL 对冲 / 启动恢复）为骨架，**全部采纳《补充意见》的工程结论**并落地。两处过时前提已校正：信号层为成熟版（schema `nrd.schema.v1.0.0`，`allow_downstream_evaluation` 存在），但**端到端接收链确实缺失**——已在执行侧补齐。

## 二、架构（单一持续主链）

`main()` → 循环 `run_cycle()`（取代旧 PLAN/ORDER 双脚本，已不再是运行入口）。每轮：

```
GetCommand → cmd_router(幂等+命令账本) → signal_receiver(降级) → gates(5门汇总)
  → [启动恢复阻塞? ] → [有持仓? manage_cycle : 急停? : 锁定? _attempt_commit : 维护推荐库]
  → 渲染「交互控制台」(置顶) + 操作提示 → _emit
```

状态/阶段：`WAIT_SIGNAL / OFFLINE_MANUAL / RECOMMEND_READY / HARD_APPROVAL_WAIT / PLAN_LOCKED /
POSITION_MANAGE / SHORT_FLAT_LONG_RESIDUAL / RECOVERY_BLOCKED / KILLED`。

## 三、交互层（用户重点，已实现）

- **状态栏交互控制台**（`display.disp_console_table`，每轮置顶）：阶段 / 执行门控 / 信号接收 / 待批方案+短确认码 / 软授权(状态+授权码) / 止盈资格 / 退出活动 / 对冲 / 预提交 / **操作提示**。
- **简化命令（短确认码）**：用户只输面板上的 4~6 位活码。`recommend` 内部仍按 `session+strategy+quality+plan_hash` 全约束匹配；质量按 S:PM 释放 0.1 **分桶**→子桶波动**不改码**（输入期间稳定），材料级变化才换码、旧码自动过期。
- **FMZ 按钮**（机器人「交互」区配置，`cmd_router` 解析）：`执行`(码)/`拒绝`/`授权止盈`(码)/`撤销授权`/`风险退出授权`(码)/`急停`/`恢复`。命令幂等键 `robot_id+session_id+refresh_seq+command_type+nonce`，消费型一次性消费、全部入命令账本。
- 验收：`GetCommand` 回测不生效，**须真实机器人空跑验收**。

## 五、《补充意见》处置（全部落地）

- **P0-1 接收链**：`signal_receiver`（schema/TTL/reject/data_quality 校验 + package_id 血缘；总线不可用→禁新开、不阻断持仓管理；OFFLINE_MANUAL 静态信号红标）。依赖（follow-up）：信号侧调 `signal_bridge` 落盘 + 原子 rename + 同托管 loopback spike。
- **P0-2 确认码/幂等**：冻结审批快照 + 一次性短确认码 + 结构化幂等键 + 漂移容差预提交（不要求实时哈希全等）。
- **P0-3 门控拆分**：`gates.py` 5 门 + 运行时急停/恢复（恢复要求重新对账+新硬批准）。
- **P0-4 四输出仲裁**：`unified_action_arbiter` preferred/executable/blocked_reason/fallback，退出受阻回退对冲。
- **P0-5 对冲 open/reduce**：`hedge_order_action` 分离，reduce_only 仅用于减仓/平仓。
- **P0-6 投影预算**：`evaluate_projected_budget` 计入拟建仓位、fail-closed。
- **P1**：风险退出授权完整语义（max_exit_spend/allowed_order_types/valid_until/revoke/consume）；面板「胜率」标注为 OTM 近似(≈1−|Δ|)；账本 9 类；显式 `SHORT_FLAT_LONG_RESIDUAL`；真实机器人空跑验收。

## 六、安全默认与上线

- 默认 5 门控全 False（全空跑）：进/退/对冲均仅展示意图与控制台，不真实下单。
- OFFLINE 无 market_context → VRP 门 fail-closed → 默认不真实成交。
- 上线顺序：真实机器人空跑回放（推荐库→`执行:码`→预提交→空跑开仓预览→`授权止盈:码`→止盈资格→退出预览→对冲预览，核对控制台/操作提示/意图表、错码/旧码拒绝、急停/恢复即时生效、同命令不重复）→ 逐门开真。
- 残留（低优先，shadow-only）：`gex_info` 增强尚未并入 realsrc（仅最新交付物 spm 单文件有；非硬门、可降级）。
